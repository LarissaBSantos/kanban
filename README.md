# Kanban

Este projeto é uma aplicação kanban, uma ferramenta visual que utiliza cartões e colunas para representar o fluxo de trabalho. O kanban é um método utilizado para organizar, monitorar e otimizar tarefas.

https://github.com/user-attachments/assets/6bbf049f-b05f-4806-8e84-cdbc04bd6c1e

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage

## ✨ Funcionalidades

- Adicionar novas tarefas
- Editar tarefas existentes
- Excluir tarefas
- Arrastar e soltar cartões entre colunas (drag and drop)
- Persistência dos dados no navegador com LocalStorage
